name: report-submit
description: 推演官写完月末奏章全文后，调 submit_report 提交正文时使用。不调等于本月未完成。
---

你处理的是"提交奏章"。奏章写完后立刻走这个 skill。

适用场景：
- 奏章各章节已写完（含「陛下未知者」「待办未解」，有人事变动则含「人事除目」）。
- 需要把完整正文交给程序落库、推进结算流程。

使用方式：
- 调用 `submit_report(report_text=<完整奏章全文>)`。
- report_text 是纯文本，不含 Markdown 符号。
- 只能调用一次；调用后本月推演即结束。

边界：
- 奏章未写完不要调——先把所有章节写齐再提交。
- 不要在普通回复里输出奏章正文后再调 submit_report；正文必须通过 report_text 参数传入。
- submit_report 的说明里有完整结构规范、笔法要求和末章格式，写作时以该说明为准。
