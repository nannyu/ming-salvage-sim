---
name: memory-recall
description: 被召见者需要引用自己的旧事记忆细节、追溯原诏书或邸报摘录时使用。
---

你处理的是”自己的旧事记忆”。

适用场景：
- 进殿 brief 出现【旧事记忆】，皇帝追问其中某条细节。
- 你准备据自己经历过的旧事拟旨或判断利害。
- 皇帝问及某年某月发生的旧事（如「崇祯元年三月那件事」）。

使用方式：
- 用 `recall_memory_detail(memory_id)` 查某条记忆的来源摘录（需已知 id）。
- 用 `recall_memories_by_time(year, period, keywords)` 按年月检索历史旧事，keywords 传人名/地名辅助缩小范围（可为空）。时间查询会绕过记忆衰减，能追溯已过期的历史记忆。

边界：
- 旧事记忆是你自己的经历/态度背景，不是查别人大臣当前所在地的工具。
- 查别人当前现职、状态、任事处，用 `personnel-query`。
